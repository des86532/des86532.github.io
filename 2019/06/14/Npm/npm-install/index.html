npm 全域安裝提示權限不足的錯誤

npm WARN checkPermissions Missing write access to /user/local/lib/node_modules

解決方法

修改 npm 所安裝目錄的權限

會要求輸入密碼，登入密碼

查看目錄是否已經切換權限

紅色區域本來會是 root ，切換權限後會變為自己主機的名稱
