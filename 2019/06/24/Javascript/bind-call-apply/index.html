bind

複製一個新函數，不會呼叫

第一個參數：設定 this 指向的物件
第二個參數: 會帶入複製 function 的第一個參數(當作固定值，不能被改變的預設參數)

call

會直接呼叫 function

第一個參數：設定this指向的物件
第二個參數：會帶入function的第一個參數

apply

會直接呼叫 function，只接受陣列作為參數

call & apply 差別只有在後面的參數， apply 要用陣列

function borrowing

function currying
