在Github上建立新專案

按下setting 裡面找到 Github Pages 的地方

點擊 Choose a theme

這是讓你選擇模板，會先以這個模板架設網站

選好模板後，按下Select theme

commit後，網站就發佈了，網址就在setting裡面可以看見
