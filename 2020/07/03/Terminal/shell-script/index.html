Shell Script主要用途就是用來協助使用者在UNIX or Linux環境上, 以更方便, 更自動化的方式來執行想要執行的指令

Vue 專案內

shell script 預設有些參數可以使用：

  • $?：表示你輸入的上個指令的狀態，一般情況下為 0，表示成功，不正常離開的不會是 0
  • $0：shell script的檔名
  • $1：表示輸入的第一個參數
  • $2：表示第二個參數

npm run BUILD_DEV：
echo $0：./deploy.sh
echo $1：dev

sh deploy.sh dev：
echo $0：deploy.sh
echo $1：dev
